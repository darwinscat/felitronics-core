# felitronics-core

**JUCE-free, real-time-safe, header-only C++20 DSP core** for the Darwin's Cat audio plugins
(TabbyEQ, OrbitCab, вҖҰ). A set of independent `process()`-RT-safe modules вҖ” each its own namespace +
CMake target, each with framework-free self-tests. **No external dependencies by default: pure C++20 standard
library** вҖ” plus one opt-in, vendored SIMD FFT backend (`fftpffft`, `-DFELITRONICS_WITH_PFFFT=ON`, default OFF).

## Modules

`core` (FFT seam В· smoother В· delay В· denormal flush) В· `eq` (Vicanek matched biquads В· SVF В· EQ engine В·
crossovers) В· `lineareq` (linear- & mixed-phase "Natural" FIR EQ over partitioned convolution) В·
`convolution` (zero-latency partitioned convolver В· click-free IR swap) В· `dynamics` В· `multiband` В·
`dynamiceq` В· `deesser` В· `stereo` В· `saturation` В· `dither` В· `oversampling` В· `limiter` (true-peak) В·
`analysis` (LUFS/LRA В· dBTP true-peak В· correlation) В· `neural` (inference seam) В·
`fftpffft` (**optional** compiled SIMD FFT backend вҖ” vendored pffft, `-DFELITRONICS_WITH_PFFFT=ON`).

Each module links on its own (`felitronics::eq`, `felitronics::lineareq`, вҖҰ) вҖ” pull only what you use.
Every module ships JUCE-free self-tests (measured audio == analytic truth).

## Build & test

```sh
cmake -S . -B build -DFELITRONICS_BUILD_TESTS=ON
cmake --build build -j
ctest --test-dir build
```

## Consume

```cmake
include(FetchContent)
FetchContent_Declare(felitronics_core
    GIT_REPOSITORY https://github.com/darwinscat/felitronics-core.git
    GIT_TAG        <pinned-tag>)
FetchContent_MakeAvailable(felitronics_core)

target_link_libraries(app PRIVATE felitronics::eq felitronics::lineareq)   # link only what you use
```
```cpp
#include <felitronics/eq/EqEngine.h>
```
